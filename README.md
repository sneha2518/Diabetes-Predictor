# Diabetes-Predictor
# 🩺 Diabetes Prediction System

A machine learning web application that predicts whether a person is likely to have diabetes based on health parameters.

---

## 🚀 Live Demo

👉 https://diabetes-predictor-1-yw6m.onrender.com/

---

## 📌 Project Overview

This project uses a Random Forest Classifier to predict diabetes based on medical attributes such as glucose level, BMI, age, and more.

The model is integrated into a Flask web application and deployed online using Render, allowing users to input their details and get instant predictions.

---

## 🧠 Features

- 🔮 Predicts diabetes risk
- 🌐 Web-based user interface
- ⚡ Fast and responsive predictions
- 📊 Uses real-world medical dataset
- 🎯 Simple and user-friendly inputs

## ⚙️ How It Works

1. User enters health details  
2. Data is sent to Flask backend  
3. Model processes the input  
4. Prediction is generated  
5. Result is displayed on the web page  

---
## 💻 How to Run Locally

1. Clone the repository:
git clone https://github.com/your-username/diabetes-predictor.git

2. Navigate to the folder:
cd diabetes-predictor

3. Install dependencies:
pip install -r requirements.txt

4. Run the application:
python app.py

5. Open in browser:
http://127.0.0.1:5000/
